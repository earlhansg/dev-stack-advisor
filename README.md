# Dev Stack Advisor

An AI-powered developer tool that recommends the right tech stack for your project. Ask a question about your app's requirements, and the    advisor fans out across three curated knowledge bases — technology profiles, project templates, and real-world case studies — then         
  synthesizes a clear, actionable recommendation via streaming.                                                                                                                                                                                                                           
  ## Tech Stack                                                                                                                                                                                                                                                                           
  | Layer     | Technology                                      |                                                                              |-----------|-------------------------------------------------|                                                                            
  | Frontend  | React 19, TypeScript, Vite 8, Tailwind CSS 4   |                                                                             
  | Backend   | Express 5, TypeScript, ts-node-dev              |                                                                            
  | AI        | OpenAI Responses API + File Search (vector stores) |

  How It Works
                                                                                                                                             
  1. The user types a question about their project requirements.
  2. The server fans out to three OpenAI vector stores in parallel:                                                                          
    - Technology profiles — frameworks, databases, mobile, CSS tools                                                                         
    - Project templates — SaaS, e-commerce, realtime collaboration, AI apps                                                                  
    - Case studies — real-world reference projects                                                                                           
  3. Results are combined into a synthesis prompt and streamed back using gpt-4.1-mini.                                                      
  4. The client renders the response token-by-token as it arrives.                                                                           
                                                                                                                                               ---                                                                                                                                        
  Important Notes                                                                                                                                                                                                                                                                         
  - The npm run setup script must be run once before starting the server. Without valid vector store IDs in server/.env, the /api/ask
  endpoint will fail.                                                                                                                          - server/.env contains sensitive credentials — ensure it is listed in .gitignore.                                                          
  - The data files in server/data/ are the source of truth for the AI's knowledge. Modify them and re-run npm run setup to update the vector 
  stores.
